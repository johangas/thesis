Network Topology

This is a demo of the Intel Galileo acting as a 6LoWPAN IoT gateway using the NetContiki border router and the Yanzi IoT-U10 or IoT-U10+ IoT devices acting as a serial 802.15.4 radio. The demos is based on the NetContiki codebase plus some open-source libraries for Javascript and Python.

Non NetContiki code used for the demo:

  • JQuery by John Resig et.al. (MIT License)
  • JQuery EasyTabs by Steve Schwartz (MIT License)
  • JQuery Modal by Kyle Fox (MIT License)
  • Javascript-surface-plot by Greg Ross (New BSD License)
  • Vis.js for Network Topology Graphs (Apache 2.0 and MIT License)
  • SimpleWebSocketServer - Python library by Dave Pallot (MIT License)
